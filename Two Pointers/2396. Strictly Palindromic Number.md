# 2396. Strictly Palindromic Number

- In an $(n-2)$-ary system, $n$ is represented as $12$, which is not a palindrome.

    <div>
        $$
        n = 1 \cdot (n-2)^1 + 2 \cdot (n-2)^0 = 12\_{(n-2)}
        $$
    </div>

```java
class Solution {
    public boolean isStrictlyPalindromic(int n) {
        return false;
    }
}
```
