# 1732. [Find the Highest Altitude](https://leetcode.com/problems/find-the-highest-altitude)

```Java
class Solution {
    public int largestAltitude(int[] gain) {
        int ans = 0, alt = 0;
        for (int h : gain) {
            alt += h;
            ans = Math.max(ans, alt);
        }
        return ans;
    }
}
```
