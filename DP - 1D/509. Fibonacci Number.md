# [509. Fibonacci Number](https://leetcode.com/problems/fibonacci-number)

```java
class Solution {
    public int fib(int n) {
        int f0 = 0, f1 = 1;
        if (n == 0) return f0;
        if (n == 1) return f1;
        int f2 = 0;
        for (int i = 2; i <= n; i++) {
            f2 = f0 + f1;
            f0 = f1;
            f1 = f2;
        }
        return f2;
    }
}
```
