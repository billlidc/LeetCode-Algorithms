# [70. Climbing Stairs](https://leetcode.com/problems/climbing-stairs)

```java
class Solution {
    public int climbStairs(int n) {
        if (n == 1) return 1;
        if (n == 2) return 2;

        int c1 = 1;
        int c2 = 2;
        int c3 = 0;

        for (int i = 3; i <= n; i++) {
            c3 = c1 + c2;
            c1 = c2;
            c2 = c3;
        }
        
        return c3;
    }
}
```


```java
class Solution {
    public int climbStairs(int n) {
        int[] dp = new int[n + 1];
        dp[0] = 1; dp[1] = 1;

        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
}
```
